## Contexte

`ensureProfiles` retourne dès qu'un seul `.env` existe. Après upgrade (v0.4.3 → v0.5.0), les 5 nouveaux profils embarqués ne sont **jamais** extraits. L'utilisateur ne voit pas les nouveaux fournisseurs.

Source : Codex adversarial high finding.

## Ce qu'il faut faire

### Étape 1 — Créer un fichier manifeste

Dans `internal/assets/`, créer `profiles-manifest.json` (embarqué) :

```json
{
  "version": "0.5.0",
  "profiles": {
    "00-claude-official.env": "sha256:abc123...",
    "01-code-pro.env": "sha256:def456...",
    "...": "..."
  }
}
```

Générer ce fichier automatiquement via `go generate` :

```go
//go:generate go run scripts/gen-manifest.go
```

### Étape 2 — Modifier `ensureProfiles`

```go
func ensureProfiles(dir string) error {
    embeddedManifest := loadEmbeddedManifest()
    installedManifest := loadInstalledManifest(dir)
    
    for name, embeddedHash := range embeddedManifest.Profiles {
        installedHash, exists := installedManifest.Profiles[name]
        
        if !exists {
            // Nouveau profil → extraire
            extractProfile(dir, name)
        } else if installedHash != embeddedHash {
            // Profil modifié dans la nouvelle version
            if userModified(dir, name, installedHash) {
                // L'utilisateur a modifié → ne pas écraser, avertir
                log.Printf("[i] Profil %s modifié localement, nouvelle version ignorée", name)
            } else {
                // Profil non modifié → mettre à jour
                extractProfile(dir, name)
            }
        }
        // else : identique → rien à faire
    }
    
    // Détecter les suppressions volontaires (tombstones)
    for name := range installedManifest.Profiles {
        if _, exists := embeddedManifest.Profiles[name]; !exists {
            // Profil supprimé de la version embarquée → créer un tombstone
            tombstone := filepath.Join(dir, "."+name+".removed")
            os.WriteFile(tombstone, []byte("removed in "+embeddedManifest.Version), 0644)
        }
    }
    
    saveManifest(dir, embeddedManifest)
    return nil
}

func userModified(dir, name, originalHash string) bool {
    content, _ := os.ReadFile(filepath.Join(dir, name))
    actualHash := sha256Hex(content)
    return actualHash != originalHash
}
```
