# Story 2.7 — Extraire `display/` de `cli/` (découplage)

**Sprint :** 🟠 2 — HIGH
**Priorité :** HIGH
**Effort :** 2h
**Agent assigné :** Forge (Dev)
**Dépendances :** Aucune
**Fichiers :** `multiai-go/internal/cli/display.go` → `multiai-go/internal/display/display.go`

---

## Contexte

`config`, `menu`, `onboarding` importent `cli` **uniquement** pour les helpers d'affichage. Ça crée un couplage artificiel "métier → orchestration".

Source : Audit architecture R1.

## Ce qu'il faut faire

### Étape 1 — Créer le nouveau package

Créer `internal/display/display.go` avec :

```go
package display

import (
    "fmt"
    "os"
    "strings"
)

var colorsEnabled = os.Getenv("NO_COLOR") == ""

func PrintSuccess(format string, args ...interface{}) {
    prefix := "[OK]"
    if colorsEnabled { prefix = "\033[32m[OK]\033[0m" }
    fmt.Fprintf(os.Stderr, prefix+" "+format+"\n", args...)
}

func PrintWarning(format string, args ...interface{}) {
    prefix := "[!]"
    if colorsEnabled { prefix = "\033[33m[!]\033[0m" }
    fmt.Fprintf(os.Stderr, prefix+" "+format+"\n", args...)
}

func PrintError(format string, args ...interface{}) {
    prefix := "[X]"
    if colorsEnabled { prefix = "\033[31m[X]\033[0m" }
    fmt.Fprintf(os.Stderr, prefix+" "+format+"\n", args...)
}

func PrintInfo(format string, args ...interface{}) {
    prefix := "[i]"
    if colorsEnabled { prefix = "\033[36m[i]\033[0m" }
    fmt.Fprintf(os.Stderr, prefix+" "+format+"\n", args...)
}

func Colorize(color, text string) string {
    if !colorsEnabled { return text }
    return color + text + "\033[0m"
}

func StatusColor(configured, total int) string { ... }
func MaskSecret(s string) string { ... }
```

### Étape 2 — Migrer les imports

Remplacer dans ces fichiers :
- `internal/config/wizard.go` : `cli.PrintWarning` → `display.PrintWarning`
- `internal/config/erase.go` : `cli.PrintSuccess` → `display.PrintSuccess`
- `internal/menu/interactive.go` : `cli.Colorize` → `display.Colorize`
- `internal/onboarding/wizard.go` : `cli.PrintInfo` → `display.PrintInfo`

### Étape 3 — Déprécier les anciennes fonctions

Dans `cli/display.go`, ajouter des wrappers dépréciés :

```go
// Deprecated: use display.PrintSuccess instead
func PrintSuccess(format string, args ...interface{}) {
    display.PrintSuccess(format, args...)
}
```

Les wrappers seront supprimés au sprint suivant.

### Étape 4 — Mettre à jour `main.go`

`main.go` peut continuer à utiliser `cli.PrintX` temporairement, ou migrer directement.
