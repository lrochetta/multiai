## Objectif

La version PowerShell avait 37 wrappers `.cmd` (un par profil). La version Go a perdu cette fonctionnalité. Générer des wrappers cross-platform (`.sh` pour Unix, `.cmd` pour Windows, `.ps1` pour PowerShell).

## Ce qu'il faut faire

### Étape 1 — Script de génération

Créer `scripts/generate-wrappers.sh` :

```bash
#!/bin/bash
# Génère les wrappers pour tous les profils

PROFILES_DIR="${MULTIAI_PROFILES_DIR:-$HOME/.multiai/profiles}"
OUTPUT_DIR="wrappers"

mkdir -p "$OUTPUT_DIR"

for profile in "$PROFILES_DIR"/*.env; do
    shortcut=$(grep "^SHORTCUT=" "$profile" | cut -d= -f2)
    if [ -z "$shortcut" ]; then
        shortcut=$(basename "$profile" .env | cut -d- -f2-)
    fi
    
    # Wrapper bash
    cat > "$OUTPUT_DIR/multiai-$shortcut" << EOF
#!/bin/bash
exec multiai launch -p "$shortcut" "\$@"
EOF
    chmod +x "$OUTPUT_DIR/multiai-$shortcut"
    
    # Wrapper cmd (Windows)
    cat > "$OUTPUT_DIR/multiai-$shortcut.cmd" << EOF
@echo off
multiai launch -p $shortcut %*
EOF
done

echo "Généré $(ls "$OUTPUT_DIR"/*.sh | wc -l) wrappers dans $OUTPUT_DIR/"
```

### Étape 2 — Ajouter au Makefile

```makefile
wrappers:
	@echo "Génération des wrappers..."
	@MULTIAI_PROFILES_DIR=internal/assets/profiles bash scripts/generate-wrappers.sh
	@echo "OK — $(shell ls wrappers/multiai-* | wc -l) wrappers générés"
```

### Étape 3 — Ajouter au PATH

Documenter dans le README :
```bash
# Ajouter les wrappers au PATH
export PATH="$HOME/.multiai/wrappers:$PATH"

# Usage direct
multiai-ds       # → multiai launch -p ds
multiai-codex55  # → multiai launch -p codex55
multiai-co       # → multiai launch -p co
```
