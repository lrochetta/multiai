## Objectif

100% des messages utilisateur sont en français. Pour un marché global, c'est bloquant. Ajouter un framework i18n minimal avec l'anglais comme première langue additionnelle.

## Ce qu'il faut faire

### Étape 1 — Créer le package `internal/i18n/`

```go
package i18n

import "os"

type Lang string

const (
    FR Lang = "fr"
    EN Lang = "en"
)

var messages = map[Lang]map[string]string{
    FR: {
        "profile.not_found":    "profil introuvable : %s",
        "profile.missing_key":  "secret obligatoire non configuré : %s",
        "config.plaintext_warn": "sera écrit EN CLAIR dans %s",
        "update.available":     "v%s disponible (vous avez %s). Lancez 'multiai update'.",
        "wizard.welcome":       "Bienvenue dans multiai ! %d profils, %d fournisseurs.",
        "wizard.configure_now": "Voulez-vous configurer vos clés API maintenant ?",
        // ... ~50 messages
    },
    EN: {
        "profile.not_found":    "profile not found: %s",
        "profile.missing_key":  "required secret not configured: %s",
        "config.plaintext_warn": "will be written IN PLAINTEXT to %s",
        "update.available":     "v%s available (you have %s). Run 'multiai update'.",
        "wizard.welcome":       "Welcome to multiai! %d profiles, %d providers.",
        "wizard.configure_now": "Configure your API keys now?",
        // ...
    },
}

func T(key string, args ...interface{}) string {
    lang := detectLang()
    tmpl, ok := messages[lang][key]
    if !ok {
        tmpl = messages[EN][key] // fallback anglais
    }
    if tmpl == "" {
        return key // clé brute si pas de traduction
    }
    if len(args) > 0 {
        return fmt.Sprintf(tmpl, args...)
    }
    return tmpl
}

func detectLang() Lang {
    if lang := os.Getenv("MULTIAI_LANG"); lang != "" {
        return Lang(lang)
    }
    if lang := os.Getenv("LANG"); lang != "" {
        if strings.HasPrefix(lang, "fr") { return FR }
    }
    return FR // défaut français (rétrocompatible)
}
```

### Étape 2 — Migrer les messages progressivement

Priorité : erreurs → menus → help → onboarding

```go
// Avant
fmt.Fprintf(os.Stderr, "profil introuvable : %s\n", name)

// Après
fmt.Fprintf(os.Stderr, i18n.T("profile.not_found", name)+"\n")
```

### Étape 3 — Traduire 50 messages clés

Liste prioritaire :
1. Tous les messages d'erreur (`internal/cli/`, `internal/config/`)
2. Menus (`internal/menu/`)
3. Onboarding (`internal/onboarding/`)
4. Commandes (`cmd/multiai/`)

### Étape 4 — Tester

```bash
MULTIAI_LANG=en multiai list
MULTIAI_LANG=en multiai config
MULTIAI_LANG=en multiai launch -p invalid  # Message d'erreur en anglais
MULTIAI_LANG=fr multiai list               # Rétrocompatibilité
```
