# Story 4.3 — Mise à jour Go 1.24 + migration yaml

**Sprint :** 🟢 4 — LOW
**Priorité :** LOW
**Effort :** 1h
**Agent assigné :** Forge (Dev)
**Dépendances :** Tous les tests doivent être verts avant
**Fichiers :** `go.mod`, `internal/catalog/catalog.go`, `internal/profile/yaml.go`, `internal/profile/project.go`

---

## Objectif

- `go.mod` déclare `go 1.22` (fin de support)
- `gopkg.in/yaml.v3` est archivé (plus de correctifs)
- Migrer vers `github.com/yaml/go-yaml` (API 100% compatible, maintenu)

## Ce qu'il faut faire

### Étape 1 — Mettre à jour go.mod

```bash
cd multiai-go
go mod edit -go=1.24
go mod tidy
```

### Étape 2 — Remplacer le module yaml

```bash
go mod edit -dropreplace gopkg.in/yaml.v3
go get github.com/yaml/go-yaml@latest
```

### Étape 3 — Mettre à jour les imports

```go
// Avant
import "gopkg.in/yaml.v3"

// Après
import "github.com/yaml/go-yaml"
```

Fichiers à modifier :
- `internal/catalog/catalog.go`
- `internal/profile/yaml.go`
- `internal/profile/project.go`

### Étape 4 — Vérifier

```bash
go mod tidy
go test ./...
go vet ./...
golangci-lint run
```

### Étape 5 — Mettre à jour la CI

```yaml
# ci.yml
- uses: actions/setup-go@<PINNED_SHA>
  with:
    go-version: '1.24'
```
