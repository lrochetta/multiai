## Actions

### Action 1 — Show HN

```markdown
Titre : Show HN: multiai — a secure multi-AI CLI router with 37 pre-configured profiles

Texte :
I built multiai because I was tired of manually exporting API keys
every time I switched between Claude Code, Codex CLI, and OpenCode.
One wrong export and your Anthropic key leaks into an OpenAI session.

multiai solves this with:
- AES-256-GCM encrypted credential store (keys never in plaintext)
- Environment isolation (~30 vars whitelist)
- Automatic fallback chains
- 37 profiles across 13 providers
- One-line install: npm install -g multiai

Built in Go, 1 dependency, Cosign-signed releases.
Feedback welcome!
```
