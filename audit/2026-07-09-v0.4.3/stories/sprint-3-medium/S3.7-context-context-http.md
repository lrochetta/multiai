# Story 3.7 — Propager `context.Context` dans les appels HTTP

**Sprint :** 🟡 3 — MEDIUM
**Priorité :** MEDIUM
**Effort :** 2h
**Agent assigné :** Forge (Dev)
**Dépendances :** Aucune
**Fichiers :** `multiai-go/internal/update/update.go`, `multiai-go/internal/openrouter/client.go`

---

## Objectif

Aucune fonction HTTP n'accepte de `context.Context`. Pas de propagation de cancellation, pas de deadlines.

Source : Audit architecture R4.

## Ce qu'il faut faire

### Étape 1 — Ajouter `ctx` aux signatures

```go
// Avant
func FetchLatestRelease() (*Release, error)
func FetchModels() (*Catalog, error)
func fetchRaw(url string) ([]byte, error)

// Après
func FetchLatestRelease(ctx context.Context) (*Release, error)
func FetchModels(ctx context.Context) (*Catalog, error)
func fetchRaw(ctx context.Context, url string) ([]byte, error)
```

### Étape 2 — Utiliser `req.WithContext(ctx)`

```go
func fetchRaw(ctx context.Context, url string) ([]byte, error) {
    req, err := http.NewRequestWithContext(ctx, "GET", url, nil)
    if err != nil {
        return nil, err
    }
    resp, err := http.DefaultClient.Do(req)
    // ...
}
```

### Étape 3 — Propager depuis `main()`

```go
func main() {
    ctx, cancel := context.WithTimeout(context.Background(), 30*time.Second)
    defer cancel()
    
    // Vérification auto-update en arrière-plan
    go func() {
        release, err := update.FetchLatestRelease(ctx)
        // ...
    }()
}
```

### Étape 4 — Gérer `ctx.Err()` dans les boucles

```go
for i := 0; i < maxRetries; i++ {
    select {
    case <-ctx.Done():
        return nil, ctx.Err()
    default:
    }
    // ... retry logic
}
```

## Tests

```go
func TestFetchLatestRelease_ContextCancelled(t *testing.T) {
    ctx, cancel := context.WithCancel(context.Background())
    cancel() // immédiatement annulé
    _, err := FetchLatestRelease(ctx)
    if err == nil {
        t.Error("expected context cancellation error")
    }
}

func TestFetchLatestRelease_Timeout(t *testing.T) {
    ctx, cancel := context.WithTimeout(context.Background(), 1*time.Nanosecond)
    defer cancel()
    time.Sleep(time.Millisecond)
    _, err := FetchLatestRelease(ctx)
    if err == nil {
        t.Error("expected timeout error")
    }
}
```
