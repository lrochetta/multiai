# Story 3.3 — `multiai update` explicite

**Sprint :** 🟡 3 — MEDIUM
**Priorité :** MEDIUM

**Agent assigné :** Forge (Dev)
**Dépendances :** S1.2, S1.3 (URL hardcodée + Cosign avant update)
**Fichier cible :** `multiai-go/cmd/multiai/cmd_update.go` (nouveau)

## Contexte

L'auto-update actuelle est silencieuse : `Check()` dans `main()` télécharge et ré-exécute le binaire sans prévenir. Comportement surprenant et potentiellement dangereux.

Source : Audit DX #5.

## Ce qu'il faut faire

### Étape 1 — Créer `multiai update`

```go
// cmd_update.go
func init() {
    register("update", cmdUpdate)
}

func cmdUpdate(args []string) error {
    currentVersion := version  // injectée par ldflags
    
    // 1. Vérifier la disponibilité
    release, err := update.FetchLatestRelease()
    if err != nil {
        return fmt.Errorf("impossible de vérifier les mises à jour : %w", err)
    }
    
    if !update.IsNewer(currentVersion, release.TagName) {
        fmt.Printf("[i] Vous avez déjà la dernière version (%s)\n", currentVersion)
        return nil
    }
    
    // 2. Afficher les infos
    fmt.Printf("Version actuelle : %s\n", currentVersion)
    fmt.Printf("Version disponible : %s\n", release.TagName)
    fmt.Printf("Taille : %s\n", humanizeBytes(release.Size))
    fmt.Println()
    
    // 3. Demander confirmation (sauf --yes)
    if !hasFlag(args, "--yes") && !hasFlag(args, "-y") {
        fmt.Print("Installer la mise à jour ? [o/N] ")
        var response string
        fmt.Scanln(&response)
        if strings.ToLower(response) != "o" && strings.ToLower(response) != "oui" {
            fmt.Println("Annulé.")
            return nil
        }
    }
    
    // 4. Télécharger et vérifier
    fmt.Print("Téléchargement... ")
    newExe, err := update.DownloadAndVerify(release)
    if err != nil {
        return fmt.Errorf("échec : %w", err)
    }
    fmt.Println("OK")
    
    // 5. Remplacer le binaire actuel
    fmt.Print("Installation... ")
    if err := update.Install(newExe); err != nil {
        return fmt.Errorf("échec : %w", err)
    }
    fmt.Println("OK")
    
    fmt.Printf("\n[OK] multiai mis à jour vers %s !\n", release.TagName)
    return nil
}
```

### Étape 2 — Ajouter les flags

```
Flags:
  --check       Vérifie sans installer (sortie JSON)
  --yes, -y     Non-interactif (pour CI/scripts)
```

### Étape 3 — Remplacer l'auto-update silencieux

Dans `main()` :

```go
// Avant (v0.4.3)
update.Check(version)

// Après (v0.5.0)
go func() {
    release, err := update.FetchLatestRelease()
    if err != nil { return }
    if update.IsNewer(version, release.TagName) {
        fmt.Fprintf(os.Stderr, "\n[i] v%s disponible (vous avez %s). Lancez 'multiai update'.\n\n",
            release.TagName, version)
    }
}()
```

## Tests manuels

```bash
multiai update --check          # Affiche version dispo sans installer
multiai update                   # Interactif : demande confirmation
multiai update --yes             # Non-interactif
multiai update                   # Déjà à jour → message
```
